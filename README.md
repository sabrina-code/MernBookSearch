# GoogleBookSearch
A React-based Google Books Search app. 

Use React components, and lifecycle methods to query and display books based on user search. Axios data fetching. 

npm start to run the program

## Technology and Skills
* React
* Node
* Express
* Google API
* Axios
* HTML5
* CSS3
* Bootstrap v4.3.1
